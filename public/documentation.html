Movie_API API Documentation

Description URL HTTP Method Query Parameters Request body data format Response body data format
Get a list of all movies /movies GET /movies None A JSON object holding titles of all of the movies
Get the data of a single movie by title /movies GET /movies/:Title None A JSON object with the genre of the specific movie
Return data about a genre by title /movies GET /movies/Genre/:Name None A JSON object with the genre of the specific movie
Return data about a director by title /movies GET /movies/director/:Name None A JSON object with the director of the specific movie.
New User Registration /users POST /users A JSON object holding data about the user to add, structured like:
{
Username: "SavageSteve",
Password: "xyz",
Email: "billiebob@gmail.com",
Birthday: "2000-01-01"
} 		A JSON object holding data about the user that was added.
Return data about a specific user by username /users GET /users/:Username None A JSON object with the director of the specific movie.
User Update Profile /users PUT /users/:Username None A JSON object holding data about the user that was added.
User Add a Movie to Favorites /users POST /users/:Username/movies/:MovieID A JSON object holding data about the movie a user wants to add to their favorites using the movie id.base
Example:
"Username":"Betty Boop",
"Password": "iAmaPassword",
"Email": "betty@gmail.com", "Favorites" : [],
"_id" : ObjectId("5db8650222a14ce2324bb438") 		A JSON object holding data about the movie that was added.
Return the list of favorite movies of the user /users GET /users/:Username/movies NONE
User Remove a Movie from Favorites /users PUT /users/:Username/movies/:MovieID NONE A text message letting the user know the movie was removed from their favorites list.
Existing User Deregister /users DELETE /users/:Username None A text message letting the user know they are no longer registered.
