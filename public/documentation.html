Movie_API API Documentation

Description URL HTTP Method Query Parameters Request body data format Response body data format
Get a list of all movies /movies GET /movies None A JSON object holding titles of all of the movies
Get the data of a single movie by title /movies GET /movies/:Title None A JSON object with the genre of the specific movie
Return data about a genre (description) by name/title (e.g., "Thriller") /movies GET /movies/Genre/:genreName None A JSON object with the genre of the specific movie
Return data about a director (bio, birth year, death year) by director name /movies GET /movies/director/:directorName None A JSON object about the director (bio, birth year, death year).
New User Registration /users POST /users A JSON object holding data about the user to add, structured like:
{
name: "SavageSteve",
favoriteMovies: ["Silence of the Lambs"],
} 		A JSON object holding data about the user that was added.
User Update username /users PUT /users/:id A JSON object holding data about the user to update username, structured like:
{
name: "Kim Updated Name",
favoriteMovies: [ ],
} 		A JSON object holding data about the user that username with updated username, structured like:
{
id: 1,
"name": "Kim Updated Name",
"favoriteMovies": []
}
User Add a Movie to Favorites /users POST /users/:id/:movieTitle None A text message letting the user know the movie was added for their favorites list.
User Remove a Movie from Favorites /users DELETE /users/:id/:movieTitle None A text message letting the user know the movie was removed from their favorites list.
Existing User Deregister /users DELETE /users/:Username None A text message letting the user know they are no longer registered.
